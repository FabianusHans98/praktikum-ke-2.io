1. Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat dan menyusun halaman web. HTML digunakan untuk menentukan struktur dari konten halaman, seperti paragraf, heading, link, gambar, dan elemen lainnya.

2. Fungsi Tag HTML

Tag HTML berfungsi sebagai penanda elemen-elemen pada halaman web. Setiap tag memiliki tujuan tertentu, misalnya <p> untuk paragraf, <h1>-<h6> untuk heading, <a> untuk tautan, dan lain sebagainya.

3. Struktur Dasar HTML

Struktur dasar HTML terdiri dari elemen-elemen seperti <!DOCTYPE html>, <html>, <head>, dan <body>. Bagian <head> berisi informasi meta, judul, dan link CSS, sedangkan <body> berisi konten yang ditampilkan.

4. Penambahan CSS dan JavaScript

CSS digunakan untuk mengatur tampilan visual halaman HTML, seperti warna, layout, dan font. JavaScript digunakan untuk menambahkan interaktivitas pada halaman web seperti validasi form, animasi, dan manipulasi DOM.

5. Contoh Penggunaan Bootstrap: Halaman Profil

Bootstrap adalah framework CSS yang sangat populer digunakan untuk mempercepat dan mempermudah pembuatan tampilan web yang rapi dan responsif. Salah satu contoh penggunaannya adalah dalam pembuatan halaman profil profesional.

Fungsinya:

  • Container dan Grid: mengatur layout agar rapi dan responsif di berbagai ukuran layar.
  • Card dan List Group: menampilkan informasi pengalaman dan kontak secara elegan.
  • Font Awesome: menambahkan ikon email dan Instagram agar tampilan lebih visual.
  • Navbar & Footer: memberikan struktur lengkap seperti halaman profil profesional modern.
